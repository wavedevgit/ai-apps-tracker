# AI Coding Apps datamining

Tracking apps like codex, claude code, openclaw and other ai coding apps versions &amp; src.

## How it works

### Open-source apps

For tools with public repositories, we just monitors their repositories directly to detect commits, releases, and code changes.

### Closed-source apps

For proprietary tools (e.g., Claude Code), we extracts and analyzes minified source code from distributed binaries. this allows us to find out information that isn't public.
